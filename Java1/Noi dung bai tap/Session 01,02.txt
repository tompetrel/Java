1. Viết chương trình java đơn giản sau (không sử dụng package):
a. Dùng một chương trình soạn thảo để tạo 1 file java với nội dung sau:
public class HelloWorld {
    public static void main(String[] args) {
        /* In dong chu Hello, World! tren standard output. */
        System.out.println("Hello World!");
    }
}
b. Biên dịch file đã tạo ở bước a thành file class.
c. Chạy file class đã tạo ở bước b trong cửa sổ MS-DOS.
